# ☕ Java Foundation

This repository documents my structured journey through Java programming.

### 1: Basics (Getting Started & Core Syntax)
- [x] Java Architecture Setup (JDK, JRE, JVM) & Hello World
- [x] Variables, Primitive Data Types & Literals
- [x] Type Conversion & Type Casting
- [x] Operators (Arithmetic, Relational, Logical)

### 2: Flow Control
- [x] Conditional Statements (If-Else, Ternary, Switch)
- [x] Loops (While, Do-While, For, Nested Loops)

### 3: Core Object-Oriented Programming (OOP)
- [x] Classes, Objects & JVM Memory layout (Stack vs Heap)
- [x] Methods & Method Overloading
- [x] Arrays(1D, multi-dimensional, Jagged, 3D)
- [x] Strings (Immutable vs Mutable, StringBuffer, StringBuilder)
- [x] Static Keyword (Variables, Methods, Blocks)
- [x] Encapsulation, Getters, Setters & `this` Keyword
- [x] Constructors & `super` Method
- [x] Naming Conventions & Anonymous Objects

### 4: Advanced OOP Concepts
- [x] Inheritance (Single & Multi-level)
- [x] Method Overriding & Dynamic Method Dispatch (Polymorphism)
- [x] Packages & Access Modifiers (Public, Private, Protected, Default)

- [x] Abstract Classes & Inner Classes (Anonymous Inner Class)

### 5: Production-Ready Java & Collections
- [ ] Exception Handling (Try, Catch, Finally, Throw, Throws, Custom Exceptions)

---

## 💻 How to Run
Each folder (package) contains standalone Java files aligned with the above phases. To run a file from the terminal:
1. Navigate to the `src` folder.
2. Compile: `javac packageName/FileName.java`
3. Run: `java packageName.FileName`

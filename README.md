# ☕ Java Foundation 

This repository documents my structured journey through Java programming.

- [ ] Variables, Primitive Data Types & Literals

- [ ] Operators (Arithmetic, Relational, Logical)

- [ ] Conditional Statements (If-Else, Ternary, Switch)
- [ ] Loops (While, Do-While, For, Nested Loops)
- [ ] Arrays (1D, Multi-dimensional, Jagged, 3D)

- [ ] Classes, Objects & JVM Memory layout (Stack vs Heap)

- [ ] Strings (Immutable vs Mutable, StringBuffer, StringBuilder)
- [ ] Static Keyword (Variables, Methods, Blocks)

---

## 💻 How to Run
Each folder (package) contains standalone Java files aligned with the above phases. To run a file from the terminal:
1. Navigate to the `src` folder.
2. Compile: `javac packageName/FileName.java`
3. Run: `java packageName.FileName`
